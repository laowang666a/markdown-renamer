#  markdown-renamer

## 📖 这个工具是什么？
**markdown-renamer** 是一个用 Python 编写的高效命令行工具，旨在帮助开发者/用户快速完成文件夹内的markdown文件的改名

## 使用示例
python main.py [-h] [-d] dir format #dir:文件夹路径 ；format：命名日期格式（比如 "%Y-%m-%d"）


## 🛠️ 安装方式
### 前提条件
* Python 3.8 或更高版本
### 通过 pip 安装 (推荐)
你可以直接通过 pip 安装到全局或虚拟环境中：
```bash
pip install requirements.txt
